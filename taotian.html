FY25 一句话 · 淘天 CMR 增长重新提速至 +6%，但增长靠"佣金率结构性切换"驱动， 代价是分部经营利润率从 45.78% 降至 43.62%——三年累计下降 2.2pct。这是"变现模式转型期"的典型特征。

0.1三大核心发现

  • CMR 增长重新提速：FY25 CMR 3,223 亿，YoY +6.0%（FY24 +4.3%），增量 +183 亿，远超 FY24 的 +125 亿。
  • 增长结构性切换：广告占比从 72% 降至 64%（绝对额三年下降 1.7%），佣金占比从 28% 升至 36%（绝对额 +42%）——典型从"广告主导"转向"广告+佣金双轮"。
  • 经营利润率结构性侵蚀：分部经营利润率三年下降 2.2pct（45.78% → 44.79% → 43.62%）——佣金虽收入贡献大，但毛利率低于广告，结构变化压制利润率。

0.2三大核心隐忧

  • 广告变现率见顶：广告收入三年从 2,099 亿微降至 2,063 亿，广告 Take Rate 仅从 2.62% 涨到 2.71%——增长引擎从广告切换到佣金，但佣金可压缩空间也在收窄。
  • GMV 持续下行：FY25 GMV 7.6 万亿，三年从 8.0 降至 7.6（-5%），用户基本盘和单客户钱包深度都在承压。
  • 利润率天花板：佣金主导的变现模型相比广告需要更多运营投入，43.6% 的利润率已是三年新低，FY26 若 GMV 继续 -2% 利润率压力将加剧。

BP 解读 · FY25 CMR +183 亿的核心驱动是"佣金率结构性提升"（+213 亿）—— 新增订单服务费、天猫类目佣金调整带动佣金占比从 28% 升至 36%。这是阿里变现模式的结构性切换： 从"广告主导"向"佣金+广告"双轮转。但代价是分部经营利润率从 45.8% 降至 43.6%—— 佣金虽收入贡献大，但毛利率低于广告，结构变化压制了利润率。

5. Driver Tree 三层穿透

Driver Tree 是 BP 分析中最强的归因工具——把 CMR 拆成 L1（AAC × GMV/buyer × Take Rate）→ L2（广告变现率 + 佣金率）→ L3（拆驱动因子）三层。

CMR 3,223 亿 ├─ 一级：CMR = AAC × 每买家GMV × Take Rate │ ├─ AAC 9.5 亿（FY23 10.3 → FY25 9.5，−0.8） ⇩ 拖累 │ ├─ 每买家GMV 8,000 元（FY23 7,767 → FY25 8,000，+233） ⇧ 提升 │ └─ Take Rate 4.24%（FY23 3.64% → FY25 4.24%，+0.60pct） ⇧ 提升 │ ├─ 二级：Take Rate 4.24% = 广告变现率 + 佣金率 │ ├─ 广告变现率 2.71%（FY23 2.62% → FY25 2.71%，+0.09pct） ⇧ 几乎不动 │ └─ 佣金率 1.53%（FY23 1.02% → FY25 1.53%，+0.51pct） ⇧⇧ 主导驱动 │ └─ 三级：广告/佣金分别驱动 ├─ 商家投放渗透率 42%（FY23 35% → FY25 42%，+7pct） ⇧ 中小商家扩面 ├─ 平均投放 31 万元/商家（FY23 28 → FY25 31，+3） ⇧ ROI 改善 └─ 类目佣金率提升 + 订单服务费 ⇧⇧ FY25 大幅提升
← 横向滑动查看完整树状结构 →
Driver Tree 总结 · Take Rate +0.60pct 的真正驱动不是"广告越收越多"，而是"佣金率结构性切换"—— 佣金率三年从 1.02% 升到 1.53%（+0.51pct），贡献 Take Rate 增量的 85%。广告变现率相对平稳（2.62→2.71%）。 这说明阿里变现路径在转：广告天花板已现，转向更稳定的"GMV 抽佣"模式。
